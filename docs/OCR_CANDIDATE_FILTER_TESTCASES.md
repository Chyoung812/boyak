# OCR 약명 후보 필터/자동완성 테스트케이스

## 목적

약봉투 OCR 원문에는 `약제비`, `영수증`, `복약안내`, `주소`, `전화번호`, `본인부담금` 같은 잡음이 섞인다. MVP에서는 Clova OCR 결과를 바로 DUR 검사에 넣지 않고 다음 순서로 처리한다.

```txt
Clova OCR 원문
→ 잡음 제거/약명 후보 필터
→ OCR 흔들림 보정
→ DUR drug_aliases top3 자동완성
→ 사용자 확인
→ 선택된 약만 DUR 검사
```

## 적용한 룰

### 1. 잡음 제거

제외 키워드 예:

```txt
약제비, 계산서, 영수증, 본인부담금, 복약안내, 주의사항, 주소, 전화, 병원, 약국, 처방, 조제, 아침, 점심, 저녁, 식후
```

### 2. 약명 후보 유지

유지 단서:

```txt
정, 캡슐, 시럽, 현탁, 과립, 액, 점안, 연고, 크림, 패치, 주사, mg, 밀리그램
```

### 3. OCR 흔들림 보정

테스트 샘플 기반 보정:

```txt
씨프로바이정250일리그럼 → 씨프로바이정250밀리그램
한미아스피린장용정100ng → 한미아스피린장용정100mg
초록소정 → 화록소정
프로바미정 → 씨프로바이정
세프로바이정 → 씨프로바이정
```

## 테스트 입력

```txt
약제비 계산서 영수증
본인부담금 2,300
복약안내
주소 서울시 어딘가
씨프로바이정250일리그럼
초록소정
한미아스피린장용정100ng
부스코판당의정
```

## 필터 결과

```txt
씨프로바이정250밀리그램
화록소정
한미아스피린장용정100mg
부스코판당의정
```

## Neon/SQLite drug_aliases 자동완성 결과

```txt
씨프로바이정250밀리그램
→ 씨프로바이정250밀리그램(시프로플록사신염산염수화물)_(0.291g/1정)

화록소정
→ 화록소정(록소프로펜나트륨수화물)_(68.1mg/1정)

한미아스피린장용정100mg
→ 한미아스피린장용정100밀리그램_(0.1g/1정)

부스코판당의정
→ 부스코판당의정(부틸스코폴라민브롬화물)_(10mg/1정)
```

## 발표용 문장

> 약봉투 OCR 결과를 그대로 사용하지 않고, 약명 패턴 기반 필터링과 OCR 오인식 보정 후, DUR 약품 별칭 DB에서 top3 후보를 자동완성합니다. 최종 DUR 검사는 사용자가 확인한 약에 대해서만 수행하여 안전성과 사용성을 함께 확보했습니다.
